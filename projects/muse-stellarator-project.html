Abstract

Accurate and time-resolved measurements of plasma parameters are critical for understanding the dynamics of magnetic confinement devices. In the MUSE compact stellarator, the need for precise diagnostics is addressed using a high-time-resolution system based on the Mirror Langmuir Probe (MLP). Unlike conventional Langmuir probes, which sweep across a wide range of bias voltages, the MLP rapidly cycles through pre-selected voltages to extract electron temperature (Te), electron density (ne), and floating potential (Vf) from a single probe tip. This rapid cycling enables spatially and temporally localized measurements, essential for studying transient events in the core, edge, and scrape-off layer (SOL) regions.

The MLP system was implemented using a field-programmable gate array (FPGA), following the approach described by Vincent et al. [1]. During this internship project, the focus was on ensuring robust control and data acquisition software, diagnosing and resolving errors, and rebuilding the code within a Linux virtual machine using the Koheron make system. The FPGA implementation was subsequently deployed to hardware, and the system was integrated into the MUSE experimental platform. These efforts establish a foundation for high-fidelity measurements of transient plasma dynamics and demonstrate the feasibility of FPGA-based, high-speed diagnostics in stellarator research.

My Contributions

  • Migrated and stabilized control software within a Linux environment using the Koheron make system for FPGA deployment and diagnostics development.
  • Implemented FPGA-based rapid bias voltage cycling for the Mirror Langmuir Probe (MLP), enabling high-time-resolution measurements of plasma parameters in the MUSE stellarator.
  • Reverse engineered and rebuilt FPGA code, reconstructed module interconnections, and resolved issues from incomplete documentation and duplicate code modules.
  • Integrated the MLP system into the MUSE experimental platform, including data acquisition, synchronization, and off-board streaming capabilities.
  • Developed and validated real-time data acquisition and control software on the Red Pitaya FPGA board, using Python for hardware communication.
  • Designed and tested high-speed data acquisition for extracting electron temperature, electron density, and floating potential, achieving microsecond-scale timing resolution.
  • Created a test bench, including a secondary Red Pitaya unit for simulated plasma current response, ensuring system behavior met expected timing and amplitude performance.
  • Improved diagnostic capabilities in the MUSE stellarator for time-resolved plasma measurements, contributing to studies of transient plasma phenomena (turbulence, blobs, edge-localized modes).
